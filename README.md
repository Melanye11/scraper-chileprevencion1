# Prevencionistas de Riesgo en Chile

Este proyecto extrae y muestra información sobre prevencionistas de riesgo en Chile desde chileprevencion.cl.

## Características

- Web scraping para obtener datos de prevencionistas
- API REST para acceder a la información
- Interfaz web para visualizar y buscar profesionales
- Filtros por región y especialidad
- Paginación de resultados
- Actualización automática de datos
